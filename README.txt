Πρόγραμμα Δημιουργίας Project: Visual Studio Code
Γλώσσα Προγραμματισμού: C++

Οδηγίες εκτέλεσης προγράμματος:

ΒΗΜΑ 1:
Θα δημιουργήσετε έναν φάκελο στον υπολογιστή σας

ΒΗΜΑ 2:
Στον φάκελο που μόλις δημιουργήσατε, θα βάλετε το αρχείο projmain.cpp που κατεβάσατε

ΒΗΜΑ 3:
Θα ανοίξετε το Visual Studio Code.

ΒΗΜΑ 4:
Θα πάτε πάνω αριστερά και θα επιλέξετε το File -> Open Folder.

ΒΗΜΑ 5:
Στο παράθυρο που θα σας ανοίξει , θα βρείτε τον φάκελο που δημιουργήσατε στο Βήμα 1 και θα τον επιλέξετε και εφόσον τον επιλέξετε , θα πατήσετε άνοιγμα φακέλου.

ΒΗΜΑ 6:
Στο Visual Studio Code θα επιλέξετε, αν δεν είναι ήδη επιλεγμένη, την πρώτη επιλογή (για εικονίδιο είναι 2 αρχεία και το 1 βρίσκετε απάνω από το άλλο) , και από δίπλα θα γράφει explorer και από κάτω θα φαίνεται ο φάκελος που δημιουργήσατε και ανοίξατε και τα περιεχόμενα του, δηλαδή το αρχείο που έχει τον κώδικα (projmain.cpp)

ΒΗΜΑ 7:
Θα επιλέξετε το αρχείο με τον κώδικα (projmain.cpp) για να το ανοίξετε και να μπορέσετε να τον δείτε.

ΒΗΜΑ 8:
Για να τον κάνετε compile θα πρέπει να πάτε προς την μέση και λίγο κάτω εκεί που λέει Terminal (το οποίο πρέπει να είναι επιλεγμένο) και να γράψετε την εξής εντολή "g++ -g projmain.cpp -o projmain.exe"

ΒΗΜΑ 9:
Εφόσον το compile γίνει , τότε θα δημιουργηθεί το εκτελέσιμο αρχείο (projmain.exe). Για να το τρέξετε θα πάτε να βρείτε τον φάκελο που δημιουργήσατε στο Βήμα 1, θα τον ανοίξετε και εκεί θα βρείτε πέρα από το αρχείο με τον κώδικα, και το εκτελέσιμο αρχείο, όπου με διπλό κλικ πάνω του θα μπορέσετε να το εκτελέσετε.



ΣΗΜΕΙΩΣΗ 1:
Η εντολή "g++ -g projmain.cpp -o projmain.exe" χρησιμοποιείται αν δεν κάνω λάθος για αυτούς που χρησιμοποιούν, όπως και εγώ, τον MinGW Compiler (ο οποίος είναι Compiler για C και C++). Αν χρησιμοποιείται κάποιον άλλον compiler δεν γνωρίζω τι πρέπει να κάνετε για να τον κάνετε compile.

ΣΗΜΕΙΩΣΗ 2 και ΠΡΟΣΟΧΗ:
Μετά από την 1η φορά της επιτυχημένης εκτέλεσης του προγράμματος, στα Έγγραφα θα δημιουργηθεί ένας φάκελος ονόματι PassGuard. Αυτόν δεν τον πειράζεται για ΚΑΝΕΝΑΝ λόγο διότι αν περάσετε δεδομένα στο πρόγραμμα και μετά είτε τον διαγράψετε είτε τον μετακινήσετε οπουδήποτε αλλού, τα δεδομένα σας αυτόματα θα καταστραφούν!
